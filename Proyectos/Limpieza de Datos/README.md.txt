# Proyecto 5: Limpieza Dataset
En este proyecto se limpio un Dataset publico extraído de la página de INEGI.


## Descripción
- **Objetivo:** Limpiar, organizar y transformar ya que el data set original muestra varios incomvenientes.
- **Fuente de datos:** Archivo CSV oficiales de INEGI en México a nivel Estatal.
- **Resultados:** 
  - Una tabla con sus respectivos datos estandarizada para la siguiente etapa de Análisis.
