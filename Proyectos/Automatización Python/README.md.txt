# Proyecto : Automatización Obtención de datos
Este proyecto ha sido creado para automatizar la obtención de datos de facturas y tablas en PDF


## Descripción
- **Objetivo:** Extraer los datos necesarios para generar una base de datos.
- **Fuente de datos:** Archivos PDF's combinados de tablas e imagenes 
- **Resultados:** 
  - Una tabla con sus respectivos datos estandarizada para la siguiente etapa de Análisis.
