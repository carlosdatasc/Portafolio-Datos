# Proyecto 3: Análisis de Ventas en Cafetería León Guanajuato
Este proyecto muestra un análisis detallado de las ventas usando Python.

## Descripción
- **Objetivo:** Identificar posibles zonas de mejora en la aplicación.
- **Fuente de datos:** Archivo CSV con datos de ventas de los últimos 4 meses.
- **Resultados:** 
  - Gran porcentaje de pedidos Pendientes en una Categoría.
  - Posibles mejoras en la aplicación.
