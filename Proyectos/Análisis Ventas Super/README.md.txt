# Proyecto 2: Análisis de Ventas
Este proyecto muestra un análisis detallado de las ventas usando PowerBI.

## Descripción
- **Objetivo:** Identificar patrones de ventas por región y categoría de productos.
- **Fuente de datos:** Archivo CSV con datos de ventas de 2022.
- **Resultados:** 
  - No hay una sucursal líder en ventas.
  - Los clientes miembros no gastan lo esperado.
